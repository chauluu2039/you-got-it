Self-esteem top-up

Once a week, list 3 things you did purely for yourself. If the list feels short, that's your sign to schedule more.

Rebalancing perfectionism

Intentionally do something you'll be bad at — a new sport, drawing, a language. Practice tolerating "good enough."

Confidence check-in

Rate your confidence 1–10 monthly. Then list one recent action that aligned with your values. Notice the connection.

Mindfulness for future-focused minds

When planning how to be better "after this," pause. Describe three things around you in the present moment.

Handling criticism

Before responding, paraphrase it back — "what I hear you saying is…" to separate feedback from emotional charge.

Taming the imagination

Capture every idea in one notebook or app. Let it live there — your mind can release it safely.

Staying grounded in details

Set a weekly recurring slot (same day, same time) for admin — finances, emails, logistics. Routine removes friction.

Locus of control

At day's end, sort events into "things I can control" vs "things I can't." Release the second column.

Watch out

Burnout creeping in: If you feel resentful toward people you're helping, or too exhausted to show empathy — stop and recharge.

Watch out

Idealism turning rigid: If you're fighting for a position more than caring about the people involved, passion has outrun compassion.

Watch out

Learned helplessness: If "why bother?" becomes frequent, build small guaranteed wins back into your days immediately.

Watch out

Isolation spiral: Withdrawing to recharge is healthy. Withdrawing to avoid an imperfect world is not — notice the difference.

Watch out

Unsolicited help mode: If you're pushing solutions on people who didn't ask, check whether you're helping them or boosting your own worth.

Watch out

Creativity without output: If ideas never leave your head, the imagination has become an escape rather than a tool.

Hosting on GitHub Pages

To go live, upload these files to a GitHub repository and enable Pages:

  1. Create a new GitHub repository (e.g. chau-dashboard)
  2. Upload index.html, manifest.json, sw.js, and both icon files
  3. Go to Settings → Pages, set source to main branch / root
  4. Your site goes live at https://yourusername.github.io/chau-dashboard
  5. On mobile, open that URL in Chrome/Safari → Add to Home Screen to install as an app

All your data stays in your browser's local storage — it never leaves your device.
